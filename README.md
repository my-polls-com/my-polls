# We The People 🇺🇸

A lightweight public opinion polling website focused on American politics.

## About
We The People is a simple, fast, and clean polling site where visitors can 
vote on current political questions and see live results instantly.
No account required.

## Features
- One vote per poll per visitor
- Live result bars with percentages
- Votes saved between sessions
- Fully responsive on mobile and desktop

## Built With
- HTML, CSS, JavaScript
- No frameworks or libraries
